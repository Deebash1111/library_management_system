A database-based Library Management System designed to manage books, members,
issue/return records, and library operations efficiently.
This project helps automate library activities such as storing book details, 
tracking issued books, and managing users.

Features

Add, update, and delete books
Manage library members
Issue and return books
Track due dates and availability
Maintain transaction records
Search books by title/author/category

 Technologies Used

SQL / Oracle Database
PL/SQL
DBMS Concepts
SQL Developer / SQL*Plus
